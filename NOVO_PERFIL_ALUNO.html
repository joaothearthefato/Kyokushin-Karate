Nova Funcionalidade - Perfil do Aluno

Instruções de implementação

📋 O que foi criado?

Uma nova página de perfil do aluno que é exibida após o registro. O aluno pode informar:

  • Idade
  • Altura (em cm)
  • Peso (em kg)
  • Faixa (selecionada do banco de dados)

🔧 Passo 1: Executar o SQL

Antes de usar a nova funcionalidade, execute o arquivo sql/adicionar_colunas.sql no phpMyAdmin:

Instruções:
  1. Abra o phpMyAdmin (http://localhost/phpmyadmin)
  2. Selecione a base de dados Oyama_Hub
  3. Clique em "SQL" na barra superior
  4. Copie e cole o conteúdo de sql/adicionar_colunas.sql
  5. Clique em "Executar"

Isso irá adicionar as colunas: idade, altura, peso e garantir que faixa_id existe.

✅ Passo 2: Testar a nova funcionalidade

Para testar:
  1. Acesse php/registro.php
  2. Preencha o formulário de registro com um novo aluno
  3. Após clicar em "MATRICULAR-SE", você será levado para a página de perfil
  4. Preencha idade, altura, peso e escolha uma faixa
  5. Clique em "Completar Perfil" para salvar

📁 Arquivos criados/modificados

  • php/perfil_aluno.php - Nova página de perfil do aluno
  • css/perfil_aluno.css - Estilos para a página de perfil (tema Kyokushin)
  • sql/adicionar_colunas.sql - Script SQL para adicionar colunas à tabela
  • php/registrar_aluno.php - Modificado para redirecionar para perfil

🎨 Características do CSS

  • Tema Kyokushin com cores: vermelho (#c62828) e ouro (#d4af37)
  • Design responsivo (funciona em mobile)
  • Campos com validação visual (focus states)
  • Gradiente de fundo similar ao resto do site
  • Botões com hover effects
  • Opção de "Pular por enquanto" para usuários que não querem completar agora

💡 Observações importantes

  • As faixas são puxadas automaticamente da tabela faixas
  • Os dados são salvos na tabela usuarios_oyama
  • O usuário pode pular o preenchimento do perfil usando o botão "Pular por enquanto"
  • O formulário valida os dados de entrada (idade, altura, peso)
